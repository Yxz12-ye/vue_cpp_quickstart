🚀 快速开始
──────────────────────────────────────────────────────────────────────────────

1. 查看可用预设:
   cmake --list-presets

2. 配置项目 (选择一个预设):
   cmake --preset win-clang-debug          # Windows Clang
   cmake --preset win-msvc-debug           # Windows MSVC
   cmake --preset linux-gcc-debug          # Linux GCC
   cmake --preset linux-clang-debug        # Linux Clang
   cmake --preset macos-clang-debug        # macOS Clang

3. 构建项目:
   cmake --build --preset win-clang-debug

4. 运行程序:
   ./build/win-clang-debug/bin/CppTemplate

🎯 核心模块
──────────────────────────────────────────────────────────────────────────────

1. Logger (日志系统)
   • 多日志级别: DEBUG, INFO, WARNING, ERROR
   • 自动时间戳
   • 可配置过滤
   • 使用: Logger::info("message")

2. StringUtils (字符串工具)
   • toUpper() / toLower()   - 大小写转换
   • trim() / trimLeft() / trimRight() - 空格修剪
   • split()                - 分割字符串
   • replace()              - 替换子串
   • startsWith() / endsWith() - 前缀/后缀检查

3. App (应用程序类)
   • 命令行参数处理
   • 版本信息显示
   • 帮助文本输出
   • 主程序逻辑入口

💡 项目自定义
──────────────────────────────────────────────────────────────────────────────
修改项目名称:
  • 编辑 CMakeLists.txt 中的 project(CppTemplate ...) 改成你的项目名

修改版本号:
  • 编辑 CMakeLists.txt 中的 VERSION 字段

修改 C++ 标准:
  • 编辑 CMakeLists.txt 中的 CMAKE_CXX_STANDARD

添加源文件:
  • 在 src/ 创建新文件
  • 更新 CMakeLists.txt 的 SOURCES 和 HEADERS 列表
  • 重新运行 cmake --preset ...
